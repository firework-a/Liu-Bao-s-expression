# 柳宝的表情
一个简单的柳宝表情生成器

## 环境要求(Windows)
1. node.js
2. http-server

## 如果尚未安装http-server,可在命令行运行以下命令以安装
### npm
~~~
npm install http-server
~~~
### pnpm
~~~
pnpm install http-server
~~~
### yarn
~~~
yarn add http-server
~~~

## 使用说明（请查看此使用说明）
1. 以管理员身份运行start.bat
2. 浏览器访问 http://127.0.0.1:8081 即可
3. 上传一张你喜欢的图片
4. 下载图片
5. 大功告成
